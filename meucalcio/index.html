    Fontes e Metodologia

    Esta calculadora visa fornecer uma estimativa do consumo de cálcio e compará-lo com recomendações gerais. Não substitui o aconselhamento de um profissional de saúde ou nutricionista.

    Fontes das Recomendações de Ingestão de Cálcio:
    • As metas de cálcio são atualmente estimadas com base em idade e sexo, utilizando valores de referência gerais inspirados nas DRIs (Dietary Reference Intakes).
    • Nota: A integração completa de peso e altura para personalizar as recomendações depende da implementação da tabela de referência específica fornecida pelo idealizador do projeto.
    Fontes dos Dados Nutricionais dos Alimentos:
    • Os dados nutricionais dos alimentos são obtidos através da TACO API (Tabela Brasileira de Composição de Alimentos), via taco-api.netlify.app. Agradecemos aos mantenedores por esta valiosa fonte de dados.
    • A Tabela TACO é uma referência confiável para alimentos encontrados no Brasil.
    Como o Cálculo de Ingestão é Feito:
    1. Perfil do Usuário: Você fornece sua idade, sexo, peso e altura.
    2. Meta Diária: Com base no seu perfil (atualmente idade e sexo), uma meta diária de cálcio é sugerida.
    3. Busca e Entrada de Alimentos: Você busca alimentos; as sugestões e dados são consultados na TACO API. Você especifica a quantidade e a unidade (g/ml) consumida.
    4. Cálculo do Cálcio por Alimento: Para cada alimento, o teor de cálcio (por 100g) é obtido da API. Este valor é usado para calcular o cálcio na porção exata que você consumiu.
    5. Total Consumido: O cálcio de todos os alimentos adicionados é somado.
    6. Resultado Visual: A barra de progresso e o status comparam o seu total consumido com a sua meta.
